# 🚀 Antigravity 懶人包：新專案套用教學指南

本指南將引導您在未來「建立全新專案」時，如何一鍵引入、安裝並調用本倉庫中的 **Antigravity 專屬自動化工作流技能 (Skills)**。

這套工作流會幫您與您的 AI 協作大腦（Antigravity）達成完美的默契，實現 **「一鍵初始化 ➡️ 開工自動同步與任務建議 ➡️ 收工自動安檢、規範提交與雲端推送」** 的全自動化開發循環。

---

## 🛠️ 第一步：在新專案中引入懶人包（支援雲端與本地離線）

當您未來在電腦中新建了一個空白資料夾，並用編輯器開啟與 Antigravity AI 助理開始對話時，您可以選擇以下任一方式引入懶人包技能：

### 🎯 方式 A：直接讀取本地檔案（離線免連網・最推薦 💾）
由於您已經將懶人包下載到本地電腦中（路徑為：`H:\我的雲端硬碟\antigravity-lazy-packs`），您可以直接叫 AI 讀取本地路徑，完全不需要連接 GitHub，速度最快且安全：

> 💬 **對 AI 助理下達的口令**：
> 「我開了一個新專案。請直接讀取我**本地電腦**上的 Antigravity 懶人包技能，本地絕對路徑是：`H:\我的雲端硬碟\antigravity-lazy-packs`，請幫我載入裡面的 `SKILL.md` 與技能設定。」

---

### 🌐 方式 B：連接 GitHub 雲端引入
如果您在其他沒有同步雲端硬碟的電腦上工作，則可以通過雲端網址直接下載：

> 💬 **對 AI 助理下達的口令**：
> 「我開了一個新專案。請幫我讀取並載入我的 Antigravity 懶人包技能，倉庫網址是：`https://github.com/andy591027/antigravity-lazy-packs`」

---

💡 **大腦載入機制**：
不論是 A 還是 B，AI 助理讀取後，都會自動識別該目錄下的 `SKILL.md` 索引，並將三大技能（`15-project-init`、`16-work-start`、`17-work-close`）自動加載至當前新專案的大腦記憶中。


---

## 🚀 第二步：三大技能實戰操作教學

一旦 AI 助理載入了技能，您就可以在接下來的開發過程中，隨時呼叫對應口令：

### 1. 🛠️ 專案 initialization（初始化）
*   **適用時機**：剛建立新專案目錄，裡面還是空的或只有少數初始檔案時。
*   **口令（對 AI 說）**：`「準備開新專案」` 或 `「初始化專案」`
*   **AI 將自動為您執行的動作**：
    1.  建立標準專案目錄結構：`src/`（原始碼）、`docs/`（文件）、`tests/`（測試）、`assets/`（靜態資源）。
    2.  自動生成標準 `.gitignore`（過濾 node_modules 等雜物）。
    3.  自動生成專案 `README.md`（包含基本介紹與架構）。
    4.  自動生成 `AGENTS.md`（AI 協作指南，讓後續的 AI 助手秒懂專案規則）。
    5.  自動建立 `task.md`（您的 TODO 任務清單檔案，用來追蹤所有進度）。
    6.  自動執行 `git init && git add . && git commit -m "chore: initial project structure"`，幫您打好 Git 本地地基。

---

### ☀️ 2. 每日「開工」狀態檢查
*   **適用時機**：每天早上或每次重新開始寫程式時。
*   **口令（對 AI 說）**：`「開工！」` 或 `「準備開始工作」`
*   **AI 將自動為您執行的動作**：
    1.  **自動防衝突同步**：在背景幫您執行 `git pull`，確保您的工作區與 GitHub 雲端代碼完全一致，100% 避免版本衝突。
    2.  **進度喚醒與智能安排**：讀取專案下的 `task.md`、`AGENTS.md` 等進度檔案。
    3.  **大腦開工報告**：AI 會在對話框中用溫和條理的繁體中文，主動向您彙報：
        *   *「您好！目前專案進度為 40%。根據 `task.md`，今日為您建議的核心任務待辦（Top 3）為：1. ... 2. ... 3. ...。請問我們今天從哪裡開始？」*

---

### 🌙 3. 每日「收工」自動備份
*   **適用時機**：今天寫得差不多，準備關電腦休息或切換任務時。
*   **口令（對 AI 說）**：`「今天寫得差不多了，收工！」` 或 `「結束今天工作」`
*   **AI 將自動為您執行的動作**：
    1.  **安全與品質檢查**：背景幫您掃描代碼是否含有敏感的金鑰（如 `API_KEY`）洩漏風險。
    2.  **自動標註 TODO**：自動將您今天在對話中口頭確認完成的任務，在 `task.md` 中標記為已完成 `[x]`。
    3.  **規範化自動提交**：自動 `git add .`，並根據您今天對話中的修改內容，智能撰寫出符合 Git 常規（Conventional Commits）的 Commit 訊息（例如 `feat: add user login component`），在本地完成 Git 提交。
    4.  **一鍵推送雲端**：自動執行 `git push` 將代碼送上 GitHub，確保進度萬無一失。
    5.  **交接備忘錄**：在對話框中寫下：**「下一次開工應繼續的起點」**，讓您下次開工時毫無斷點。

## 💡 貼心提示：如何讓 AI 運作得更完美？

-   **大腦本地快取**：當您在同一個工作電腦上開發時，Antigravity 會自動快取您的技能。若您在新專案中直接喊 `「開工！」`，AI 通常就能直接辨識並執行。
-   **雙腦協同**：如果您將新專案也與 [Obsidian](file:///H:/%E6%88%91%E7%9A%84%E9%9B%B2%E7%AB%AF%E7%A1%AC%E7%A2%9F/antigravity-lazy-packs/README.md) 第二大腦綁定，AI 在「開工」與「收工」時，還會自動讀取與寫入您的 Obsidian 日誌，讓您的知識網與代碼進度完美同步！
